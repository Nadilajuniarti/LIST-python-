# List Dasar Python

# Apa itu List?
List adalah struktur data yang digunakan untuk menyimpan banyak data dalam satu variabel.
Contoh:
buah = ["apel", "jeruk", "mangga"]
print(buah)

# Ciri-ciri List
-Menggunakan tanda []
-Dapat menyimpan banyak data
-Bisa berisi berbagai tipe data
-Bersifat mutable (bisa diubah)

# Index
Index adalah posisi data dalam list dan dimulai dari 0
contoh:
buah = ["apel", "jeruk", "mangga"]
print(buah[0])   # apel
print(buah[-1])  # mangga

# Operasi Dasar
-Menambah Data
buah.append("pisang")
-Mengubah Data
buah[1] = "anggur"
-Menghapus Data
buah.remove("apel")
